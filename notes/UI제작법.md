## 참고 유튭 영상
https://www.youtube.com/watch?v=6iq6CjUJvxc&t=79s
위 영상을 통해 디자인을 뽑아내는 과정을 간략히 살펴볼 수 있다.
1) 리서치와 카피 잡기
2) Variant 로 디자인 잡기
3) Vairant 가 만들어준 디자인을 유심히 보고, 꼬리에 꼬리를 무는 체이닝 방식으로 디자인 시작
4) 디자인 선택 후 우측 상단 ... 누르고 New Chat from desing 클릭. 이때 클로드 코드로
만든 카피를 하나씩 던져주면서, 이 디자인 시스템 하에서 이런 카피 섹션을 만들어줘
이렇게 요청을 보내면 같은 디자인 틀 안에서 작업할 수 있게 됩니다. 
5) 상단 open in 을 클릭하면 claude code 옵션이 있습니다. 결과로 나온 섹션 모두를 이런
식으로 내보내기를 하고, 그것들을 클로드코드에게 붙여넣고, 이 디자인 코드를 기반으로 
전체 디자인 시스템을 먼저 만들어 달라고 요청합니다. 요청할 때, 본인의 프로젝트 기반에 
맞게 요청을 같이 해주면 됩니다. Next.js, Tailwind css 반응형 등의 요청을 말이죠.
